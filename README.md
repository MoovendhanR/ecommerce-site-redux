## Run Locally

```bash
  git clone https://github.com/MoovendhanR/ecommerce-site-redux
```

Go to the project directory

```bash
  cd ecommerce-site-redux
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```
